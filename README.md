# Akashic Playground

[website](https://akashic-games.github.io/playground/#/edit/default)

# development

## build

```sh
npm run build
```

## watch build

```sh
npm run dev
```

# deploy

以下のコマンドで gh-pages 上にデプロイできます。

```sh
npm run deploy
```
